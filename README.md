# Vanshika Shah Portfolio Website

This is a GitHub Pages-ready personal portfolio website.

## Files
- `index.html` — main scrollable website
- `style.css` — styling and responsive layout
- `assets/` — add your profile photo, project images, and resume here

## How to use with GitHub Pages
1. Create a new GitHub repository named something like `portfolio` or `vanshika-shah`.
2. Upload `index.html`, `style.css`, and the `assets` folder.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Your site will be live at:
   `https://YOUR-GITHUB-USERNAME.github.io/REPOSITORY-NAME/`

## Replace placeholders
- Add your resume as: `assets/Vanshika_Shah_Resume.pdf`
- Replace the profile image placeholder by adding an image and editing the hero card in `index.html`.
- Replace project image placeholders with screenshots or diagrams.
